## 🔭 Selected Projects

### 🛰️ SIESPRO — IoT Perimeter Safety System
LoRa SX1278 master–slave network with AutoACK for perimeter monitoring. Perimeter-crossing classification with **Random Forest**, real-time dashboard via **FastAPI + Supabase**.

### 💡 LightWell — Portable Spectral Analyzer
AS7262-based CCT and melanopic lux estimator. ESP32-C3 → Supabase pipeline, **local LLM inference (Ollama)** via FastAPI, Node.js frontend.

### 🌱 Agrometeorological LoRa System
Long-range environmental monitoring (up to 1.2 km) with LoRa SX127x + Yagi–Uda antennas. ESP32 with DHT11/BMP280 for real-time field data logging.

### 🎬 Tech Outreach — Quantum & AI Education
Educational videos on AI and Quantum Computing with **12,000+ views**. Technical scripts + editing to make complex topics accessible.

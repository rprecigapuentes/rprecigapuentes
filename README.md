# Hi, I'm **rprecigapuentes** 👋

Electronics engineering student at the **Universidad Nacional de Colombia**, passionate about bringing **intelligence to physical systems**.  
My work combines **hardware/software co-design**, **digital system verification**, and **TinyML for embedded intelligence**.  
I believe in the power of smart hardware — devices that can sense, learn, and adapt efficiently.

---

## Areas of Interest

- **HW/SW Co-design:** modeling and verification of architectures in **SystemC**, IP integration, and performance testing.  
- **Embedded Systems:** development with **FPGAs**, **microcontrollers**, and IoT sensors.  
- **Embedded AI & Digital Twins:** implementation of **GRU/LSTM** neural models for predictive control and system dynamics.  
- **Wireless Communication:** **LoRa** networks, Yagi-Uda antennas, and low-power protocols.  
- **Technical Outreach:** public and academic dissemination about AI, hardware, and embedded systems.

## Featured Projects

| Project | Description |
|----------|-------------|
| [**Agrometeorological-LoRa-System**](https://github.com/rprecigapuentes/Agrometeorological-LoRa-System) | Agrometeorological monitoring system using **LoRa** and custom **Yagi-Uda** antennas. Collects environmental data in rural areas and transmits over long distances. Implemented in **C++**. |
| [**GCD_HW_SW_SystemC**](https://github.com/rprecigapuentes/GCD_HW_SW_SystemC) | Hardware/software co-design of Euclid’s GCD algorithm in **SystemC**, including verification and performance comparison versus a pure software version. |
| [**psoc-trafficlight-4color-udb**](https://github.com/rprecigapuentes/psoc-trafficlight-4color-udb) | **Four-color traffic light** implemented in **PSoC Creator 4.4** using Universal Digital Blocks (UDBs). Includes reusable Blinky and controller modules. |
| [**raspberry-oximeter-prolog**](https://github.com/rprecigapuentes/raspberry-oximeter-prolog) | Smart oximeter with triage logic implemented in **Python** and **Prolog**, using **Raspberry Pi**, **MAX30100** pulse sensor, and **DHT11** temperature sensor. |
| [**systemc-bus-arbitration**](https://github.com/rprecigapuentes/systemc-bus-arbitration) | **SystemC** study of bus arbitration policies comparing **fixed priority** vs **round-robin**, with performance metrics like utilization and contention. |
| [**Van-de-Vusse-Digital-Twin**](https://github.com/rprecigapuentes/Van-de-Vusse-Digital-Twin) | Digital twin of a non-isothermal Van de Vusse reactor. Integrates **MATLAB** simulations with **GRU/LSTM** neural models to predict reactor dynamics. |
| [**tech-outreach**](https://github.com/rprecigapuentes/tech-outreach) | Public and academic presentations on hardware, AI, and embedded systems. Includes materials for scientific and educational outreach. |
